User Notes

You can add simple textual notes to a transcript. User notes are displayed using an italic font and are marked with the icon.

User notes are not automatically removed when they have reached the configured expiry age.

Adding a Note

To add a user note:

  1. Select Add Note (key equivalent ⌘+) from the Transcript menu.

    Alternatively, click the Add Note button in the main window toolbar (use View > Customize Toolbar to add the Add Note item):

  2. The Note panel is displayed.
  3. Enter the note text into the field provided.
  4. Press the Add to Transcript button to add the note to the transcript.

The Note panel is not closed when the Add to Transcript button is pressed allowing multiple notes to be added.

Editing a Note

To edit a user note:

  1. Select the note in the outline view.
  2. Click the note again to begin editing.
  3. Change the note text as required.
  4. Press Return or click outside the editing field to save changes.

Removing Notes

To remove notes from a transcript:

  1. Select the entries to remove in the transcript view.
  2. Select Delete from the File menu (key equivalent ⌘⌫) or press the delete button in the toolbar.
