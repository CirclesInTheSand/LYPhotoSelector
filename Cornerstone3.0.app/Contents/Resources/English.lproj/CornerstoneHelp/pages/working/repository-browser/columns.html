Repository Browser Columns

  • Name

    The name of the file or folder.

  • Recent Modification

    Recently-modified files and folders are displayed using one of the following indicators:

    Modified today Modifications to the file or folder have been committed today.
    Modified this week Modifications to the file or folder have been committed since the start of the current calendar week.

  • Date Modified

    The date at which the file or folder was last modified. This corresponds to the date at which changes to the file or folder were last committed.

  • Size

    The size of the file.

  • Kind

    Textual description of the type of the file.

  • Revision

    The item’s current revision, i.e. the revision at which changes were last committed to the repository. For folders this is also the revision at which changes were last committed to an item located in the folder or one of its sub-folders.

  • Property Indicator

    Items (files or folders) which have Subversion properties are marked with a icon.

  • Lock Status

    Files which are locked are marked with a icon.

  • Author

    The name of the author that committed the item’s latest revision to the repository.

    The author will be empty for revisions committed to the repository using an anonymous http:// connection.
