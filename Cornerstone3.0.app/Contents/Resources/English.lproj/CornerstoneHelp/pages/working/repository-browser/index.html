The Repository Browser

The repository browser displays a familiar, Finder-like interface for manipulating a repository’s files.

A repository’s browser is displayed by selecting the repository from the source list.

Staying Up-to-Date

The repository browser shows the files in the repository’s HEAD revision at the time the browser was first displayed.

Using Cornerstone to commit changes to a repository will cause the repository browser to be automatically refreshed.
