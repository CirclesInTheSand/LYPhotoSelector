Alternate Menu Functions

Cornerstone includes a number of functions which do not differ enough from the standard function to warrant a dedicated menu item or toolbar button.

Instead, the function is assigned a different key equivalent and is only displayed in the menu when the key equivalent’s modifier is pressed.

Menu alternates are activated by holding down a key modifier such as Option (⌥) or Shift (⇧) with the menu displayed.

The alternate functions are always available via their key equivalents. For example, Working Copy > Commit All Changes… is always available using ⌥⌘T.
